## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte de uma APS (Atividade Prática Supervisionada) com foco em sustentabilidade e conscientização ambiental.

A aplicação calcula emissões anuais considerando:

- Consumo de energia elétrica
- Uso de carro
- Viagens aéreas
- Transporte público
- Produção de resíduos
- Tipo de dieta

---

## ⚙️ Funcionalidades

- ✅ Cálculo de emissões por categoria
- ✅ Total anual em kg e toneladas de CO₂e
- ✅ Estimativa de créditos de carbono
- ✅ Estimativa de árvores para compensação
- ✅ Exportação de resultados em CSV
- ✅ Validação de entrada do usuário
- ✅ Testes automatizados com pytest
